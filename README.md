# School Artificial Intelligence Governance Repository

## Overview

This repository contains the school's Artificial Intelligence (AI) governance framework.

It provides a central, version-controlled location for managing AI policies, guidance, risk assessments and governance decisions.

The repository exists to ensure that AI is adopted within the school in a way that is:

- Safe.
- Ethical.
- Transparent.
- Educationally valuable.
- Aligned with relevant guidance and legislation.

---

# Purpose

Artificial Intelligence is becoming increasingly important in education, employment and society.

The school recognises that AI provides opportunities to:

- Enhance teaching and learning.
- Improve accessibility.
- Support creativity.
- Increase efficiency.
- Develop future digital skills.

However, AI must be introduced responsibly. The school must consider:

- Data protection.
- Safeguarding.
- Academic integrity.
- Equality.
- Accuracy.
- Security.

This repository provides the framework for managing those considerations.
